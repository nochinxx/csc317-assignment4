Projects

JavaScript Calculator

JavaScript · HTML · CSS

This project is a responsive web-based calculator built with HTML, CSS, and JavaScript. It supports chained operations, percentage calculations, decimal input, and sign toggling (+/−), functioning similarly to a modern mobile calculator. The interface uses CSS variables and the clamp() function to scale button and text sizes for consistent usability across devices, with a height-aware rule preventing scrolling on short displays such as the Nest Hub. A dark color scheme enhances readability, while invalid operations like division by zero return “Error” for mathematical accuracy. Repeated IDs were replaced with class selectors to maintain valid, standards-compliant HTML and CSS.
